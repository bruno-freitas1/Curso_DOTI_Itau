Programa Gama Academy de Treinamentos



O que é?

O programa gama academy foi criado para transformar as pessoas, focado na aplicação de recursos para o desenvolvimento profissional.
O Curso é altamente hands-on, ampliando o significado da aplicação das ferramentas para o ensino pautado na andragogia.


Quem pode participar?

Colaboradores que estão dispostos a sair da zona de conforto e desenvolver novas habilidades.


Quais cursos são oferecidos?

  • Gama Empreendedorismo
  • Agile Academy
  • Programa de Desenvolvimento DOTI DEV
